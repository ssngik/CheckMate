## 소셜로그인 복구중. 소셜로그인 방식 변경
  - 회원가입 한 적이 없는 사용자가 소셜 로그인 시도
  - USER-001 응답을 통해 회원가입 한 적이 없는 유저란걸 파악
  - 닉네임 설정 화면으로 이동 후 회원가입 시도
  - 회원 가입 API 호출이 성공하면 (200 응답), (화면을 리다이렉트하지 않고)내부적으로 로그인 API 호출
